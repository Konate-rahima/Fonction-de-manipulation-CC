# Fonction de manipulation CC

## Description
Ce projet contient plusieurs fonctions JavaScript :

- Manipulation de chaînes de caractères
- Manipulation de tableaux
- Fonctions mathématiques

## Fonctions incluses
- reverseString
- countCharacters
- capitalizeWords
- findMax / findMin
- sumArray
- filterArray
- factorial
- isPrime
- fibonacci
